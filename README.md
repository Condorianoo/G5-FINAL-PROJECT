ABSTRACT:
This project proposes a centralized, web-based Lost & Found Management System tailored for Cabuyao City to address the inefficiencies of current social media-based reporting. While local Facebook groups are widely used (e.g., the “MMCL - New Students” Facebook group previously named “MMCL - Lost & Found” during the A.Y. 2024-2025 as a joke referencing the sudden influx of Lost & Found items being reported), their unstructured nature often leads to critical information being buried by unrelated content. This system aims to provide a dedicated public service portal where citizens and local officials (e.g., School Security, Government Staff) can report missing items in a structured format. Developed using ASP.NET Core and SQL Server, the platform aims to replace the fragmented social media feeds with searchable, organized databases, significantly improving the retrieval rate of lost property through a more streamlined pipeline.
