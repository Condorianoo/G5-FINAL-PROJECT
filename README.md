# Cabuyao *Official* Lost & Found System

## Project Overview
The Cabuyao Official Lost & Found Recovery Portal is a centralized digital solution designed to bridge the gap between people who have lost belongings and those who have found them within the City of Cabuyao. By replacing the high-noise, low-structure of social media “Lost & Found” groups with Relational Data Models, it makes information (e.g., descriptions, photos, locations) easier to access and verifiable by local administrators. This ensures that a lost item found within Cabuyao remains searchable and actionable until the moment it is legally returned to its owner.

## Installation Steps
1. **Clone the Repository**
    - Open a terminal and run the following commands:
    ```
    $ git clone https://github.com/Condorianoo/G5-FINAL-PROJECT.git
    $ cd G5-FINAL-PROJECT
    ```

2. **Open the Solution**
    - Open the folder containing the cloned GitHub repository
    - Run the `.sln` file inside the cloned repository

3. **Build & Run the Project**
    - In Visual Studio, press `F5` and let the application launch in your browser
